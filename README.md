The BeestBot 🤖🦿

📌 Project Description

BeestBot is an Arduino UNO R3-based walking robot inspired by Theo Jansen’s Strandbeest mechanism. The project replicates lifelike locomotion using a mechanical linkage system that converts rotational motion into efficient walking steps. Designed to demonstrate the integration of mechanical design, embedded systems, and robotics, BeestBot is capable of obstacle detection and wireless control using Bluetooth communication.

The robot combines Arduino-based motor control, ultrasonic sensing, and a Theo Jansen linkage structure to achieve stable and efficient movement over different surfaces. This project highlights practical applications of kinematics, sensor interfacing, and hardware prototyping while showcasing how engineering concepts can be used to mimic natural motion.

---

🛠️ Components Used

- Arduino UNO R3
- Motor Driver Module
- BO Dual Shaft Gear Motors (60 RPM)
- Servo Motor
- HC-SR04 Ultrasonic Sensor
- Bluetooth Module (HC-05/HC-06)
- MDF Board Chassis
- Breadboard and Connecting Wires
- Li-ion Battery and 9V Battery
- RGB LEDs
- Switch

---

⚙️ Working Principle

1. The Arduino UNO acts as the main controller for the robot.
2. Gear motors drive the crank mechanism connected to the Theo Jansen linkage.
3. The linkage converts rotational motion into coordinated walking steps.
4. The ultrasonic sensor continuously measures the distance to obstacles.
5. When an obstacle is detected, the robot can stop or change direction.
6. Bluetooth communication enables wireless control through a smartphone.
7. The servo motor assists in movement control and directional adjustments.
